# Music

All tracks in this folder were downloaded from OpenGameArt for prototype use.

- `Contemplation.mp3` - main exploration ambience. Author: Joth. License: CC0. Source: https://opengameart.org/content/contemplation-0
- `LostInABadPlace.ogg` - darker ambience for Cat Vision or distorted spaces. Author: congusbongus. License: CC0. Source: https://opengameart.org/content/lost-in-a-bad-place-horror-ambience-loop
- `ProjectUtopia.ogg` - eerie relaxed loop candidate. Author: congusbongus. License: CC0. Source: https://opengameart.org/content/project-utopia-seamless-loop
- `CalmTrackLoop.ogg` - calm exploration loop candidate. Author: pmiller. License: CC0. Source: https://opengameart.org/content/calm-track
- `HeavenlyLoop.ogg` - soft memory/warm-place loop candidate. Author: isaiah658. License: CC0. Source: https://opengameart.org/content/heavenly-loop

All tracks are CC0/Public Domain, so attribution is not required, but author credits are kept here for clarity and gratitude.
